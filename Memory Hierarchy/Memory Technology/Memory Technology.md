### 메모리 기술
-----
1. 오늘날 사용되는 메모리 계층구조에서는 네 가지 주요 기술이 사용
   - 메인 메모리는 DRAM(Dynamic Random Access Memory)으로 구현
   - 프로세서에 더 가까운 계층인 캐시에는 SRAM(Static Random Access Memory)이 사용
     + DRAM은 SRAM보다 훨씬 느리지만 비트당 가격이 덜 비싸면, DRAM은 메모리 비트당 면적이 작기 때문에 가격 차이가 생기게 되고, 같은 양의 실리콘으로 더 큰 용량을 만들 수 있음
   - 플래시 메모리는 비휘발성 메모리로, 개인 휴대용 기기에서 2차 메모리로 사용
   - 자기 디스크는 서버에서 가장 크고 가장 느린 계층을 구현하는 데 사용됨

2. 접근 시간과 비트당 가격은 기술에 따라 매우 다양
