-----
### 플래시 메모리
-----
1. 플래시 메모리는 전기적으로 지울 수 있고 프로그래밍이 가능한 ROM(Electrically Erasable Programmable Read-Only Memory, EEPROM)의 한 종류
2. 디스크나 DRAM과 달리 플래시 메모리의 쓰기는 비트를 마모시킴
   - 이 단점을 극복하기 위해 대부분 플래시 제품은 여러 번 쓰기가 수행된 블록을 덜 사용된 블록에 재사상(Remapping)해서 쓰기를 분산시키는 컨트롤러를 사용
   - 이 기법을 마모 균등화(Wear Leveling)라고 부름
   - 마모 균등화를 사용하므로 개인의 휴대용 기기가 플래시의 쓰기 한계를 넘는 경우는 거의 없음

3. 마모 균등화가 플래시의 잠재적 성능을 떨어뜨리긴느 하지만, 상위 계층 소프트웨어가 각 블록의 마모를 관리하지 않는 한 마모 균등화를 사용할 수 밖에 없음
4. 마모 균등화를 수행하는 플래시 컨트롤러는 제조 과정에서 잘못된 메모리 셀을 사용하지 않게 함으로써 수율을 향상시킬 수 있음
