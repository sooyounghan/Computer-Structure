-----
### 네트워크 위상 구현
-----
: 고려해야 할 사항
   - 링크 길이는 통신 속도를 빠르게 하는 데 드는 비용에 영향을 미침
     + 일반적으로 길이가 길수록 고속 통신에 더 많은 비용이 빌셍
     + 선이 짧아지면 필요한 구동 전력이 작아도 되므로 길이가 짧을수록 한 링크에 더 많은 전선을 할당할 수 있고, 당연히 가격도 싸짐

   - 칩이 2차원 구조이므로 3차원 위상을 2차원으로 변환해야 함
   - 에너지도 고려해야 함
     + 예를 들어, 멀티코어 칩이 단순한 격자 위상을 갖는 이유는 에너지를 고려한 결과일 수 있음
