-----
### 결론
-----
1. 서비스로서의 소프트웨어(SaaS)의 중요성이 커지고 있으며, 이러한 서비스를 제공해주는 데 클러스터가 매우 성공적인 방법으로, 지리적으로 분산된 데이터센터를 비롯한 상위 게층의 여유분 제공이 이러한 서비스를 1년 365일 하루 24시간 전 세계 고객에게 제공할 수 있게 해줌
2. 창고 규모 컴퓨터(WSC)가 서버 설계의 목표와 원칙들을 바꾸고 있음
3. SIMD와 벡터 연산은 포스트 PC 시대에 큰 역할을 담당하고 있는 멀티미디어 응용과 잘 어울림 : 전통적인 병렬 MIMD 프로그래밍보다 프로그램하기가 더 쉽다는 이점도 있고, 에너지 측면에서도 MIMD보다 더 효율적
4. 모든 데스크톱과 서버 마이크로프로세서 제작사들은 고성능을 얻기 위해 멀티프로세서를 만들고 있음
5. 멀티프로세서는 프로세서 개수에 선형적으로 비례하는 속도 개선을 성공적으로 정의
   - 이는 n개의 프로세서를 구매하거나 유지하는 비용이 프로세서 하나의 n배라고 가정하는 것
