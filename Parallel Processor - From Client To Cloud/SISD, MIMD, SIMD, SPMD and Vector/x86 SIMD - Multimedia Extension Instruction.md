### x86의 SIMD : 멀티미디어 확장 명령어
-----
1. 크기가 작은 정수형 데이터에 대한 서브워드 병렬성은 1996년에 소개된 x86 마이크로프로세서의 멀티미디어 확장 명령어 MMX가 나온 동기가 됨
2. Moore의 법칙대로, 더 많은 명령어들이 추가되어 스트리밍 SIMD 확장(Streaming SIMD Extension, SSE)으로, 최근에는 개선된 벡터 확장(Advanced Vector Extension, AVX)으로 발전
   - AVX는 64비트 부동 소수점 연산 8개를 동시에 처리할 수 있음
3. 연산과 레지스터 데이터 크기는 멀티미디어 명령어의 opcode에 표시
4. 레지스터와 연산의 데이터 크기가 커지면서 멀티미디어 명령어 개수도 폭발적으로 증가하여 현재는 수백 개의 SSE 명령어와 AVX 명령어들이 제공되고 있음
